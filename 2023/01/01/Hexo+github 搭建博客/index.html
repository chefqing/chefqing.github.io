Hexo+github 搭建博客

1.环境需要：nodejs,git

2.连接github

3.创建SSH 密钥

3.1 输入 ssh-keygen -t rsa -C "GitHub 邮箱"，然后一路回车

3.2 进入 [C:\Users\用户名.ssh] 目录（要勾选显示“隐藏的项目”），用记事本打开公钥 id_rsa.pub 文件并复制里面的内容。

登陆 GitHub ，进入 Settings 页面，选择左边栏的 SSH and GPG keys，点击 New SSH key。

Title 随便取个名字，粘贴复制的 id_rsa.pub 内容到 Key 中，点击 Add SSH key 完成添加。

3.3 验证连接

打开 Git Bash，输入 ssh -T git@github.com 出现 “Are you sure……”，输入 yes 回车确认。

4. 创建GihubPages 仓库

命名为 username.github.io

username 为github 用户名

5. 安装Hexo

5.1 使用npm 安装hexo

npm install -g hexo-cli

5.2新建一个文件夹存放Hexo 程序文件

mkdir hexo_blog

5.3 初始化并安装

要在空的目录下（上面的hexo_blog）

hexo init

npm install

5.4 启动本地服务器进行预览

访问 http://localhost:4000，出现 Hexo 默认页面，本地博客安装成功！

6. 部署hexo 到GithubPages

6.1 安装hexo-deployer-git

npm install hexo-deployer-git –save

6.2 修改_coinfg.yml 文件末尾的Deployment 部分

6.3 hexo 将网站上传到GitHubPage
